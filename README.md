# Investment Journal

상장 기업에 대한 투자 분석을 기록하는 저장소입니다.

## 사용 방법

### Claude Code에게 투자 분석 요청하기

Claude Code를 사용하여 간단하게 투자 분석 문서를 생성할 수 있습니다.

**기본 요청 방법:**

```
{회사명} 투자 분석 문서 작성해줘
```

**예시:**

```
코인베이스 투자 분석 문서 작성해줘
```

```
테슬라(TSLA) 투자 분석해줘
```

```
엔비디아 회사에 대한 투자 분석 문서를 작성해줘
```

### 생성되는 문서

Claude Code는 자동으로:
- 웹 검색을 통해 최신 정보 수집
- 표준화된 7개 섹션 구조로 문서 작성
- `YYYY-MM-DD-{회사명}.md` 형식으로 파일 저장
- Git commit 및 push

### 문서 구조

각 투자 분석 문서는 다음 섹션을 포함합니다:

1. **헤더** - 회사명, 티커, 주가, 시가총액
2. **회사 개요** - 사업 분야, 제품/서비스, 최근 이벤트
3. **재무 분석** - 실적, 재무 비율, 가이던스
4. **사업 현황** - 고객, 경쟁사, 시장 포지션
5. **투자 포인트** - 강점, 리스크, 성장 가능성
6. **참고 자료** - 웹 검색 출처
7. **면책 조항**

## 면책 조항

이 저장소의 모든 문서는 정보 제공 목적으로만 작성되었으며, 투자 권유가 아닙니다. 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.
